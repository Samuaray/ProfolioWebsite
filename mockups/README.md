# Website Mockup Gallery

This folder contains 4 different futuristic and premium website mockups to help you explore various design styles, color schemes, fonts, and aesthetics for your portfolio website.

## How to View

Simply open `index.html` in your web browser to see all mockups in a gallery view, or open any individual mockup file directly.

## Mockup Designs

### 1. Cyberpunk Neon (`cyberpunk-neon.html`)
**Style:** Dark, vibrant, and edgy
- **Color Scheme:**
  - Dark navy background (#0a0e27)
  - Neon cyan (#00ffff) and magenta (#ff00ff) accents
  - Gradient text effects
- **Fonts:**
  - Orbitron (headings) - futuristic and bold
  - Rajdhani (body) - clean and modern
- **Key Features:**
  - Animated grid background
  - Glowing neon effects
  - Gradient buttons with shine animations
  - Perfect for tech/gaming/creative portfolios

### 2. Minimal Luxury (`minimal-luxury.html`)
**Style:** Elegant, sophisticated, and premium
- **Color Scheme:**
  - Deep blue gradients (#1a1a2e → #0f3460)
  - White and light gray accents
  - Subtle gradient overlays
- **Fonts:**
  - Playfair Display (headings) - elegant serif
  - Inter (body) - clean sans-serif
- **Key Features:**
  - Glassmorphism-inspired cards
  - Generous white space
  - Subtle hover animations
  - Perfect for professional/creative/design portfolios

### 3. Glassmorphism (`glassmorphism.html`)
**Style:** Modern, trendy, and vibrant
- **Color Scheme:**
  - Vibrant gradient background (purple → pink)
  - Frosted glass effect with blur
  - Semi-transparent white overlays
- **Fonts:**
  - Poppins - friendly and modern
- **Key Features:**
  - Backdrop blur effects
  - Floating animated shapes
  - Semi-transparent glass cards
  - Perfect for modern/creative/app portfolios

### 4. High-Tech Grid (`high-tech-grid.html`)
**Style:** Technical, precise, and developer-focused
- **Color Scheme:**
  - Pure black background (#050505)
  - Matrix green (#00ff88) primary
  - Electric blue (#0088ff) secondary
- **Fonts:**
  - JetBrains Mono - monospace/terminal font
  - Space Grotesk - modern geometric
- **Key Features:**
  - Animated grid patterns
  - Scanline effect overlay
  - Terminal-style design elements
  - Angled card corners
  - Perfect for developer/tech/engineering portfolios

## Design Elements Comparison

| Feature | Cyberpunk | Minimal Luxury | Glassmorphism | High-Tech |
|---------|-----------|----------------|---------------|-----------|
| Background | Dark animated grid | Gradient | Vibrant gradient | Pure black grid |
| Primary Color | Neon cyan | White | White/transparent | Matrix green |
| Typography Style | Bold futuristic | Elegant serif | Friendly modern | Monospace tech |
| Animation Level | High | Low | Medium | High |
| Vibe | Edgy & vibrant | Sophisticated | Fresh & modern | Technical |

## Tips for Choosing

- **Cyberpunk Neon**: Choose if you want to stand out with bold colors and animations
- **Minimal Luxury**: Choose if you prefer elegance and sophistication
- **Glassmorphism**: Choose if you want modern, on-trend design
- **High-Tech Grid**: Choose if you're a developer and want a technical aesthetic

## Customization Ideas

You can mix and match elements from different mockups:
- Use glassmorphism cards with the cyberpunk color scheme
- Apply the high-tech grid background to the minimal luxury layout
- Combine the luxury typography with neon accent colors
- Add scanline effects to any design for extra tech feel
